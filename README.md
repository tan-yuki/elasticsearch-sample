elasticsearch-sample
====================

## Premise

- node (>= v0.8.23)
- npm (>=1.2.18)
- elasticsearch (>= v0.90.1)
  - localhost:9200
  
## First step

1. edit `twitter_steam.js` and fill your some twitter keys.
1. boot local elasticsearch
1. execute `index_mapping.sh`
1. execute this command:
  
  ```bash
  npm install
  ```
